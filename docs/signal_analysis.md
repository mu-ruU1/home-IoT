# 赤外線信号の解析

Panasonic 製エアコンのリモコンは家製協(AEHA)[フォーマット](http://elm-chan.org/docs/ir_format.html)が使用されています

## 暖房・冷房・除湿

> 温度: 20℃, 風量: 自動, 風向: 自動  
> として固定する

### 1. 暖房

```json
{
  "format": "AEHA",
  "data": [
    [2, 32, 224, 4, 0, 0, 0, 6],
    [2, 32, 224, 4, 0, 73, 40, 128, 175, 13, 0, 14, 224, 0, 0, 134, 0, 4, 43]
  ]
}
```

### 2. 冷房

```json
{
  "format": "AEHA",
  "data": [
    [2, 32, 224, 4, 0, 0, 0, 6],
    [2, 32, 224, 4, 0, 57, 40, 128, 175, 13, 0, 14, 224, 0, 0, 134, 0, 4, 27]
  ]
}
```

### 3. 除湿

```json
{
  "format": "AEHA",
  "data": [
    [2, 32, 224, 4, 0, 0, 0, 6],
    [2, 32, 224, 4, 0, 41, 50, 128, 175, 13, 0, 14, 224, 0, 0, 134, 0, 4, 21]
  ]
}
```

### 4. 除湿弱

```json

  "data": [
    [2, 32, 224, 4, 0, 0, 0, 6],
    [2, 32, 224, 4, 0, 41, 50, 127, 175, 13, 0, 14, 224, 0, 0, 134, 0, 4, 20]
  ]
}
```
